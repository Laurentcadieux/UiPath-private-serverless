# UiPath Private Serverless

Private serverless project workspace for UiPath automation experiments, deployment notes, and OpenClaw/OMX orchestration.

## Purpose

- Build private, local-first serverless endpoints for UiPath automation workflows.
- Keep public exposure disabled unless explicitly approved.
- Use UiPath references and skills when designing UIP/Uipath automation work.
- Keep deployment scripts, docs, source, and tests separated.

## Layout

- `src/` - application or endpoint source code
- `deploy/` - deployment scripts, infrastructure notes, packaging output
- `docs/` - architecture, runbooks, and UiPath design notes
- `tests/` - smoke tests and regression checks

## Current Status

- Project scaffold created.
- OMX project guidance initialized with `omx agents-init`.
- No public endpoint is exposed.
